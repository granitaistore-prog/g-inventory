# Age Estimator — Android-додаток для оцінки віку по обличчю

Простий Android-додаток: відкриває камеру, знаходить обличчя і показує
**приблизний** вік. Написаний на Kotlin, використовує:
- **CameraX** — робота з камерою
- **ML Kit Face Detection** — пошук обличчя в кадрі
- **TensorFlow Lite** — нейромережа, яка й оцінює вік

⚠️ **Важливо розуміти:**
- Оцінка віку по фото завжди **приблизна** (похибка зазвичай ±5–10 років),
  це не медичний і не юридичний спосіб визначення віку.
- Все обчислення відбувається **локально на телефоні** — фото нікуди не
  надсилається (у цьому коді немає інтернет-запитів).

---

## Крок 0. Що вам знадобиться

1. Комп'ютер (Windows/Mac/Linux) — саму збірку на телефоні зробити не вийде,
   потрібен Android Studio на комп'ютері.
2. Безкоштовна програма **Android Studio**: https://developer.android.com/studio
3. USB-кабель для підключення Samsung A24 до комп'ютера (або емулятор,
   але камера краще працює на реальному телефоні).
4. ~30–60 хвилин часу для першої збірки (Android Studio довантажує компоненти).

---

## Крок 1. Встановіть Android Studio

1. Завантажте й встановіть Android Studio з офіційного сайту (посилання вище).
2. Запустіть його один раз і дайте йому завершити початкове налаштування
   (Android Studio Setup Wizard) — він сам довантажить потрібний SDK.

---

## Крок 2. Відкрийте цей проєкт

1. Розпакуйте архів з проєктом (папка `AgeEstimatorApp`) у будь-яке місце,
   наприклад `Документи/AgeEstimatorApp`.
2. Відкрийте Android Studio → **File → Open** → виберіть папку `AgeEstimatorApp`.
3. Дочекайтесь, поки внизу закінчиться напис **"Gradle sync"** (перший раз
   це може тривати кілька хвилин — Android Studio довантажує залежності).
4. Якщо Android Studio запропонує оновити Gradle/Plugin — можна погодитись.

---

## Крок 3. Модель для визначення віку (ОБОВ'ЯЗКОВО)

Сам код уміє *використовувати* нейромережу, але файл готової навченої
моделі (`age_model.tflite`, кілька мегабайт) у проєкт не включений —
це окремий бінарний файл, який неможливо вкласти в текстовий код.

Що зробити:

1. Знайдіть у пошуку **GitHub** (github.com) відкриті проєкти на кшталт
   "age gender estimation tflite android" — таких є декілька, зазвичай
   у них у папці `assets` вже лежить готовий файл `.tflite` для оцінки
   віку (часто разом зі статтю/моделлю, натренованою на датасеті UTKFace
   або IMDB-WIKI).
2. Завантажте файл моделі (зазвичай має назву на кшталт `age_model.tflite`
   або `model_age_gender.tflite`).
3. Перейменуйте його на **`age_model.tflite`** і покладіть у папку:
   ```
   AgeEstimatorApp/app/src/main/assets/age_model.tflite
   ```
4. Видаліть текстовий файл-нагадування у цій же папці
   (`ПОКЛАДІТЬ_СЮДИ_age_model.tflite.txt`).
5. Відкрийте файл `app/src/main/java/com/example/ageestimator/AgeEstimator.kt`
   і перевірте константи вгорі класу:
   ```kotlin
   private const val INPUT_SIZE = 64   // розмір зображення, який очікує модель
   ```
   Якщо на сторінці моделі вказано інший розмір входу (наприклад 200x200
   або 224x224) — підправте це число відповідно до опису моделі.

Якщо цей крок пропустити — додаток все одно запуститься й покаже
камеру, але замість віку виведе повідомлення, що модель не знайдена.

---

## Крок 4. Підключіть Samsung A24 і встановіть додаток

1. На телефоні: **Налаштування → Про телефон** → кілька разів натисніть
   на "Номер збірки", доки не з'явиться повідомлення "Ви стали розробником".
2. Далі: **Налаштування → Параметри розробника** → увімкніть
   **"Налагодження USB" (USB debugging)**.
3. З'єднайте телефон з комп'ютером USB-кабелем, на телефоні підтвердіть
   запит "Дозволити налагодження USB?".
4. В Android Studio вгорі має зʼявитися назва вашого пристрою (Samsung SM-A24…)
   у списку пристроїв.
5. Натисніть зелену кнопку **Run ▶** (або Shift+F10).
6. Android Studio збере додаток і встановить його прямо на телефон.
7. На телефоні дозвольте доступ до камери, коли додаток запитає.

Готово — можна наводити камеру на обличчя й натискати "Визначити вік".

## Крок 5. (Опційно) Викласти проєкт на GitHub

Якщо хочете саме мати його на GitHub (для збереження/показу коду):

1. Створіть акаунт на https://github.com, якщо ще немає.
2. Натисніть **New repository**, дайте назву, наприклад `age-estimator`.
3. В Android Studio: **VCS → Import into Version Control → Share Project on GitHub**
   (або через термінал: `git init`, `git add .`, `git commit -m "init"`,
   потім підключити віддалений репозиторій і `git push`).

GitHub тут потрібен лише як сховище коду — сам додаток на телефон
встановлюється через Android Studio (Крок 4), а не "з GitHub" напряму.

---

## Можливі проблеми

- **"Gradle sync failed"** — перевірте інтернет-з'єднання, спробуйте
  **File → Sync Project with Gradle Files**.
- **Телефон не бачить Android Studio** — перевстановіть драйвер Samsung
  USB або спробуйте інший USB-кабель/порт.
- **"Обличчя не знайдено"** — перевірте освітлення, тримайте обличчя
  ближче до камери, обличчя повинно бути в кадрі повністю.
- **Модель видає нереалістичний вік** — це нормально для приблизних
  моделей; спробуйте іншу натреновану модель або перевірте, що
  `INPUT_SIZE` в коді збігається з очікуваннями конкретної моделі.
